# Jersey Store

Jersey Store es un e-commerce de camisetas de fútbol desarrollado como proyecto full stack.  
El objetivo del proyecto es construir una tienda online completa, con identidad visual propia, catálogo de productos, carrito de compras, sistema de usuarios, pedidos y dashboard de administración.

## Identidad de marca

**Nombre comercial:** Jersey Store  
**Rubro:** tienda online de camisetas de fútbol  
**Eslogan:** Donde la pasión por el fútbol se viste con estilo.  
**Público objetivo:** fanáticos del fútbol, coleccionistas de camisetas y usuarios interesados en indumentaria deportiva moderna.

## Paleta de colores

- Negro: estética moderna y premium.
- Rojo: pasión, energía e intensidad futbolera.
- Blanco: contraste y buena legibilidad.
